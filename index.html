Trading Data Pipeline

Cloud-enabled Data Engineering pipeline for financial market data

Built a data engineering pipeline that extracts, validates, transforms and analyzes financial market data using Python, Pandas, PySpark, Azure Blob Storage and Databricks.

The pipeline processes multiple ETF datasets, calculates daily returns, moving averages, volatility and ATR, stores the processed data in CSV and Parquet formats, uploads Parquet files to Azure Blob Storage, and continues the analysis in Databricks using Apache Spark and Delta Lake.

The project also includes structured ETL steps, configurable tickers and dates, automated data validation, logging and unit tests with pytest.

Python | Pandas | PySpark | Azure Blob Storage | Databricks | Delta Lake | Parquet | PyArrow | yfinance | pytest

Heritage Housing Issues

Machine Learning project for house price prediction

Built a machine learning application to predict residential property prices in Ames, Iowa. The project combines exploratory data analysis, feature engineering, predictive modeling and an interactive Streamlit dashboard for data visualization and price predictions.

Developed a complete machine learning workflow including data cleaning, preprocessing, feature engineering, hyperparameter optimization with GridSearchCV, model evaluation and deployment. Multiple regression models were compared before selecting the final Random Forest model.

Achieved an R² score of 0.87 on unseen test data using an optimized Random Forest Regressor, providing accurate house price predictions through a deployed Streamlit application.
